> FrontEnd em React para o projeto Proffy, que visa conectar alunos e professores para agendamento de aulas. <br />
> Desenvolvido na NLW#2 da Rocketseat.

## Install

```sh
yarn install
```

## Usage

```sh
yarn run start
```

## Run tests

```sh
yarn run test
```
