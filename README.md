> Desenvolvido na NLW#2 da Rocketseat.

-----------------------------------------------------
## Clone the Repository
```sh
git clone https://github.com/Macahyba/proffy-web.git
```

## Install

```sh
cd proffy-web
yarn install
```

## Usage

```sh
yarn start
```
